Knowie

2019
Individual Project
UI | Motion Design
Adobe Illustrator · Figma · After Effects

Knowie is a crowdsourcing Q&A platform that enables people with experience and expertise in different areas to answer each other's questions by having a one-on-one conversation. With great computational powers, Knowie will be able to connect the user in need with a good match to answer the question, and index the record for further references. Knowie comes in the form of a desktop device and a mobile app, allowing the user to seek consultation and provide help at any time and place. I designed the visual system, the model for the Knowie bot, and the user interface for the app.
